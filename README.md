# AI Campus Buddy

A Flask-based web application for providing instant feedback on student assignments using AI.

## Features

- Upload assignment text for evaluation
- Get overall evaluation, score, strengths, and areas for improvement
- Language clarity analysis
- Responsive web interface

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables (optional for Hugging Face API)
4. Run the app: `python app.py`

## Usage

Open http://127.0.0.1:5000 in your browser and paste your assignment text for feedback.

## Technologies

- Flask
- Hugging Face Transformers (optional)
- TextStat for readability analysis
- HTML/CSS/JavaScript for frontend
